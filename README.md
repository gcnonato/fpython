> Para usar os scripts que precisam de senhas faça o seguinte:
- Crie um arquivo **.env** na raiz da pasta, no caso, fpython
- Copie o conteúdo abaixo e preencha com suas informações:
~~~~

INSTA_PASSWORD=

SPOTIFY_EMAIL=
SPOTIFY_SENHA=

RDS_DB_NAME=

RDS_PASSWORD=
